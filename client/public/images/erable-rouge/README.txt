IMAGES À AJOUTER POUR L'ÉRABLE ROUGE
======================================

Photos recommandées pour l'érable rouge (Acer rubrum) :

1. FLORAISON (Mars-Avril) - PRIORITÉ HAUTE
   - erable-rouge_fleurs_01.jpg : Fleurs rouges avant les feuilles
   - erable-rouge_fleurs_02.jpg : Gros plan des glomérules floraux
   
2. VUE GÉNÉRALE
   - erable-rouge_vue_generale_01.jpg : Arbre en fleurs (printemps)
   - erable-rouge_vue_generale_02.jpg : Port arrondi caractéristique
   - erable-rouge_vue_generale_03.jpg : Arbre en couleurs automnales (ESSENTIEL!)
   
3. FEUILLAGE
   - erable-rouge_feuilles_01.jpg : Feuilles vertes (été)
   - erable-rouge_feuilles_02.jpg : Feuilles lobées détail
   - erable-rouge_feuilles_03.jpg : Couleurs automnales rouge écarlate (PRIORITÉ!)
   - erable-rouge_feuilles_04.jpg : Progression des couleurs automnales
   
4. FRUITS
   - erable-rouge_fruits_01.jpg : Samares rouges (mai-juin)
   - erable-rouge_fruits_02.jpg : Samares ailées en V
   
5. RAMEAUX/TRONC
   - erable-rouge_rameaux_01.jpg : Rameaux rouges et bourgeons rouges (hiver)
   - erable-rouge_tronc_01.jpg : Écorce gris-argenté

NOTES :
- Les couleurs automnales sont LA caractéristique principale à photographier !
- Idéalement photographier en septembre-octobre (couleurs rouges spectaculaires)
- La floraison précoce rouge (mars-avril) est aussi très intéressante
- Format JPG recommandé
- Résolution minimale : 1200x800 pixels
