IMAGES À AJOUTER POUR L'ÉRABLE DU JAPON
========================================

Photos recommandées pour l'érable du Japon (Acer palmatum) :

1. FEUILLAGE - PRIORITÉ TRÈS HAUTE
   - erable-japonais_feuilles_01.jpg : Feuilles palmées finement découpées (ESSENTIEL !)
   - erable-japonais_feuilles_02.jpg : Détail découpe feuilles (5-9 lobes)
   - erable-japonais_feuilles_03.jpg : COULEURS AUTOMNALES rouge écarlate (PRIORITÉ !)
   - erable-japonais_feuilles_04.jpg : Variété pourpre (si disponible)
   - erable-japonais_feuilles_05.jpg : Feuillage lacinié finement découpé
   
2. VUE GÉNÉRALE - PRIORITÉ HAUTE
   - erable-japonais_vue_generale_01.jpg : Port sculpté élégant
   - erable-japonais_vue_generale_02.jpg : Silhouette automnale (rouge/orange)
   - erable-japonais_vue_generale_03.jpg : En jardin japonais (contexte)
   
3. FLORAISON
   - erable-japonais_fleurs_01.jpg : Fleurs pourpre-rouge discrètes (avril-mai)
   
4. RAMEAUX/SILHOUETTE HIVER
   - erable-japonais_rameaux_01.jpg : Ramification fine dense (hiver)
   - erable-japonais_rameaux_02.jpg : Silhouette sculpturale hivernale
   
5. FRUITS
   - erable-japonais_fruits_01.jpg : Samares rougeâtres (automne)

NOTES :
- Le FEUILLAGE DÉCOUPÉ est LA caractéristique à photographier !
- Couleurs automnales SPECTACULAIRES (rouge/pourpre/orange)
- Privilégier variétés pourpres si disponible (Atropurpureum, Bloodgood)
- Photographier idéalement en automne (septembre-octobre)
- Silhouette hivernale aussi très décorative
- Petit arbre (4-8m) parfait petits jardins
- Format JPG recommandé
- Résolution minimale : 1200x800 pixels
