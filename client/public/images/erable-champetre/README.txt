IMAGES À AJOUTER POUR L'ÉRABLE CHAMPÊTRE
==========================================

Photos recommandées pour l'érable champêtre (Acer campestre) :

1. VUE GÉNÉRALE - PRIORITÉ HAUTE
   - erable-champetre_vue_generale_01.jpg : Arbre entier (port arrondi)
   - erable-champetre_vue_generale_02.jpg : En alignement ou haie
   
2. FEUILLAGE - PRIORITÉ HAUTE
   - erable-champetre_feuilles_01.jpg : Feuilles 5 lobes arrondis (caractéristique)
   - erable-champetre_feuilles_02.jpg : Détail face/dos feuilles
   - erable-champetre_feuilles_03.jpg : Couleurs automnales jaune doré
   
3. FRUITS - PRIORITÉ MOYENNE
   - erable-champetre_fruits_01.jpg : SAMARES HORIZONTALES (180°) - TRÈS CARACTÉRISTIQUE !
   - erable-champetre_fruits_02.jpg : Gros plan samares (ailes opposées)
   
4. FLORAISON
   - erable-champetre_fleurs_01.jpg : Fleurs jaune-vert (avril-mai)
   - erable-champetre_fleurs_02.jpg : Floraison mellifère
   
5. RAMEAUX/TRONC
   - erable-champetre_tronc_01.jpg : Écorce gris-brun fissurée
   - erable-champetre_rameaux_01.jpg : Crêtes liégeuses ailées (si présentes)

NOTES :
- Les SAMARES HORIZONTALES sont LA caractéristique unique à photographier !
- Feuilles plus petites que autres érables (4-10cm)
- Couleurs automne : jaune doré (pas rouge comme érable rouge)
- Arbre indigène français - BON choix Bessancourt (tolère calcaire)
- Format JPG recommandé
- Résolution minimale : 1200x800 pixels
